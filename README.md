# Bataille Navalle

## Description

Ce projet est une implémentation en Java du jeu classique Bataille navale (aussi appelé « Touché-Coulé »), inspirée des règles traditionnelles. Dans ce jeu, chaque joueur place secrètement une flotte sur une grille, puis à tour de rôle tente de deviner les positions des navires adverses pour les toucher ou les couler. :contentReference[oaicite:1]{index=1}

Le dépôt contient le code source dans le dossier `src`, ainsi qu’un diagramme UML dans le dossier `UML`. :contentReference[oaicite:2]{index=2}

## Fonctionnalités

- Mise en place d’une flotte — chaque joueur place ses navires sur une grille (orientation verticale ou horizontale).  
- Gestion des tirs — l’attaquant indique des coordonnées (ex : `B5`), puis le jeu indique si c’est « dans l’eau », « touché » ou « coulé » selon le cas. :contentReference[oaicite:3]{index=3}  
- Tour de jeu alterné — les joueurs jouent à tour de rôle.  
- Interface console — le jeu s’exécute via terminal, sans interface graphique.
